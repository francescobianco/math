# Benvenuto 👋

Questo è un **wiki matematico**: ogni documento è un file Markdown salvato nella
stessa repository che pubblica il sito. La repo è sia l'applicazione sia il
contenitore di tutti gli oggetti del wiki.

## Come funziona

- Premi **+ Nuovo** per creare un *paper*, un *libro* o delle *note*.
- Finché non premi **Salva** il documento resta una **bozza** in memoria, non scritta su disco.
- Premendo **Salva** viene creato un file `.md` dentro `content/` e l'albero a sinistra si aggiorna.

## Math inline e in display

Puoi scrivere matematica inline come $e^{i\pi} + 1 = 0$ oppure in display:

$$\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}.$$

Buona scrittura!
