# Welcome 👋

This is a **math wiki**: every document is a Markdown file stored in the same
repository that publishes the site. The repo is both the application and the
container for all of the wiki's objects.

## How it works

- Press **+ New** to create a *page*, *book*, *paper*, or *note*.
- Until you press **Save**, the document stays a **draft** in memory, not written to disk.
- Pressing **Save** commits a `.md` file into `library/` on GitHub (you provide a token once per session), and the tree on the left updates.

## Inline and display math

You can write inline math like $e^{i\pi} + 1 = 0$ or in display:

$$\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}.$$

Happy writing!
