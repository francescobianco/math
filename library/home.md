# Math Wiki 🏠

Welcome to the wiki **home**. This page is always present and is the first one
you see on startup: use it as an index and a starting point.

## What you'll find here

- 📃 **Pages** — stable content like this home page.
- 📖 **Books** — collections organized into chapters.
- 📄 **Papers** — single articles or proofs.
- 📝 **Notes** — quick jottings.

Pick a document from the tree on the left, or press **+ New** to create one: it
stays a draft until you press **Save**, which commits the file straight to the
GitHub repository (you are asked once for a token, kept for the session only).
